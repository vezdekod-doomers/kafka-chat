# Сборка и запуск
- в src/main/resources/application.properties в параметре `kafka.bootstrapAddress` указать адрес кафка-брокера
- открыть проект в Intellij Idea Ultimate 2020.3.3 (есть триальная версия), импортировать gradle-проект, запустить Hack1Application
- в Edit Configuration поставить галочку Parallel Run, чтобы можно было запустить зразу несколько инстансов приложения
- можно чатиться
